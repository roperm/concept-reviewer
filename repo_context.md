# Repository Context

This repository contains requirements for a system called **Concept Reviewer**.

The system reviews manufacturing process concepts using a panel of expert agents and produces a structured PDF review report.

The repository currently contains requirement documents only. The implementation must be generated from these documents.

## Key Requirement Files

experts.md  
Defines expert roles and responsibilities.

input.md  
Defines how the user provides concept input.

output.md  
Defines the structure of the expert review output and PDF report.

architecture_principles.md  
Defines architectural constraints and system design rules.

## Implementation Goal

Generate a full working system including:

- GPT frontend
- backend API
- expert agents
- panel chair agent
- orchestration engine
- PDF generation
- schemas
- repository structure
